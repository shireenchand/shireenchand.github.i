Description

ARMDET is developed using PyTorch, utilizing the YOLOv5 model, is a sophisticated solution designed to identify guns and knives present in images, videos, and real-time streams. This system, deployed using Flask and Streamlit, holds immense importance in enhancing public safety and providing an extra layer of security to people in various environments.

The YOLOv5 model, built upon PyTorch, is a state-of-the-art object detection algorithm that demonstrates remarkable accuracy and real-time performance. By training this model on a diverse dataset of guns and knives, the system becomes capable of accurately recognizing these weapons in different scenarios.

Deployed using Flask and Streamlit, the system provides a user-friendly interface for accessing its functionality. Flask, a web framework, serves as the backbone for hosting the application, handling incoming requests, and managing the communication between the user and the system. Streamlit, on the other hand, offers a convenient way to visualize and interact with the results, making it easy for users to input images, videos, or live streams for weapon detection.

Upon analyzing the provided visual data, the system utilizes the trained YOLOv5 model to detect the presence of guns and knives. This detection process involves examining the input frame by frame and drawing bounding boxes around the identified weapons. The system then determines the confidence level associated with each detection, providing an indication of the model's certainty regarding the presence of a weapon.

One of the critical features of this system is its ability to send notifications to the respective authorities upon detecting a weapon. This functionality is achieved using Twilio, a powerful communication platform. By integrating Twilio's capabilities, the system can instantly notify the relevant authorities, enabling them to take immediate action and ensure public safety.

The importance of this project lies in its potential to prevent acts of violence, minimize response times during emergencies, and enhance overall security. By automating the detection process, the system reduces the reliance on human operators and minimizes the chances of human error or oversight. This capability is especially valuable in environments where surveillance is essential but continuous human monitoring may not be feasible or efficient.

Moreover, the system can be deployed in various settings, such as transportation hubs, schools, public spaces, and security installations. Its versatility allows it to adapt to different scenarios, providing an extra layer of protection against potential threats involving weapons.
