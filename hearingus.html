Hearing Us

Description

Hearing Us is a remarkable application that harnesses the power of state-of-the-art technology to bridge the communication gap between individuals who use American Sign Language (ASL) and those who rely on spoken language. By combining computer vision, natural language processing (NLP), and machine learning techniques, this system provides real-time translation of sign language gestures into written sentences, enabling seamless communication between people using different modes of expression.

At its core, the system utilizes the I3D model, one of the state-of-the-art models in the field of video classification and action recognition. Trained on the WLASL (Word-Level American Sign Language) Dataset, the I3D model has gained an exceptional ability to understand and interpret sign language gestures accurately. With a vocabulary consisting of 2000 common words used in ASL, this system covers a broad range of concepts, allowing for meaningful and comprehensive communication.

To ensure a smooth user experience, the system incorporates various technologies and frameworks. Python is the primary programming language, providing a versatile and efficient environment for development. OpenCV, a popular computer vision library, is employed to capture and process video input, enabling the system to recognize and track sign language gestures in real-time. PyTorch, a deep learning framework, supports the deployment of the I3D model for efficient inference.

To enhance the translation process and generate coherent and contextually appropriate sentences, the system leverages NLP-based APIs. These APIs analyze the predicted words from the I3D model and generate meaningful sentences that accurately represent the intended message. A Rewriter-Paraphraser API may be utilized to refine and improve the sentence structure, ensuring clarity and conciseness.

To make the system accessible to users, it is deployed as a web application using Flask, a popular web framework in Python. By utilizing Ngrok, the system can be easily accessed remotely, enabling communication across different devices and locations. Streamlit, a framework for building interactive web applications, provides an intuitive interface.

The impact and importance of this sign language translation system are profound. It addresses a significant barrier that individuals who are deaf or hard of hearing face in their daily lives by facilitating effective communication with the broader community. By seamlessly translating sign language gestures into written sentences in real-time, the system enables people who rely on ASL to engage in conversations, access information, and participate more fully in various social, educational, and professional settings.
